# 🍽️ LUXE Dining

A modern luxury restaurant web application built with **React**, **TypeScript**, **Vite**, **Redux Toolkit**, and **Tailwind CSS**.

LUXE Dining delivers a premium online dining experience with a beautiful interface, fast performance, and a fully responsive design.

---

## ✨ Features

- Luxury modern UI
- Responsive Design
- Fine Dining Menu
- Advanced Food Categories
- Search & Filtering
- Dish Details
- Shopping Cart
- Wishlist
- Online Checkout
- Table Reservation
- Customer Reviews
- Authentication
- User Dashboard
- Order History
- Admin Dashboard
- Dark Mode
- Fast Performance

---

## 🚀 Installation

```bash
npm install
```

Run development server

```bash
npm run dev
```

Production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
